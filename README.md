## 🕹️ 플레이 방법

### 🌐 웹 브라우저 (PC)

`StroopFruitGame.html` 파일을 브라우저에서 열면 바로 플레이!

| 입력 | 동작 |
|------|------|
| 마우스 클릭 | 색깔 버튼 선택 |
| 키보드 `1` ~ `6` | 왼쪽부터 순서대로 버튼 선택 |
| `Space` / `Enter` | 게임 시작 / 재시작 |

### 📱 안드로이드 (APK)

```bash
# 빌드
export ANDROID_HOME=/path/to/android-sdk
./gradlew assembleDebug

# APK 위치
app/build/outputs/apk/debug/app-debug.apk
```

1. APK를 폰으로 전송
2. **출처를 알 수 없는 앱 설치 허용** (설정 → 보안)
3. 설치 후 실행!
